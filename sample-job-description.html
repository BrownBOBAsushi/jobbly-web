About the Role

We are looking for a Senior Frontend Engineer with expertise in React, TypeScript, and Next.js. You will work on building scalable web applications and collaborate with cross-functional teams to deliver exceptional user experiences. This role offers the opportunity to lead frontend architecture decisions and mentor junior developers.

Key Responsibilities

  • Design and develop responsive, high-performance web applications using React and Next.js
  • Write clean, maintainable TypeScript code following best practices
  • Collaborate with designers, product managers, and backend engineers to deliver features
  • Optimize applications for maximum speed and scalability
  • Mentor junior developers and conduct code reviews

  • Stay up-to-date with emerging frontend technologies and trends

  • Experience with RESTful APIs and GraphQL
  • Strong problem-solving and debugging skills
  • Excellent communication and teamwork abilities
