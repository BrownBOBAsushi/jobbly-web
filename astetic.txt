## OVERALL DESIGN PHILOSOPHY

Create a calm, confident, and effortlessly modern interface that balances "Stripe meets Notion" aesthetics - clean minimalism with just enough personality to feel human, not corporate. The design should feel like a breath of fresh air: spacious, uncluttered, and trustworthy.

Core principles:
- Zen Minimalism: Generous whitespace, no visual clutter, every element serves a clear purpose
- Soft Gradients: Purple-to-pink gradient accents create warmth without overwhelming
- Approachable Professionalism: Friendly but serious - this is about careers, not games
- Subtle Depth: Soft shadows and gentle curves suggest layers without heaviness
- Confident Simplicity: "We're so good, we don't need to shout"

## CRITICAL DESIGN NOTES

**DO:**
- Use generous whitespace - don't be afraid of empty space
- Keep borders subtle (rgba opacity 0.1-0.2)
- Apply gradients sparingly - only for key elements
- Make hover states obvious but smooth
- Ensure all interactive elements have clear focus states
- Use the purple-pink gradient ONLY for: hero text, primary buttons, and brand elements

**DON'T:**
- Overcrowd the interface - one concept per section
- Use harsh shadows - keep them soft and purple-tinted
- Apply gradients everywhere - solid colors for most UI
- Make borders too prominent - they should whisper, not shout
- Forget hover/active states on interactive elements
- Use pure black (#000000) - always use warm dark grays
